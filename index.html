About Me

I'm Pavel Kamarashev — a DevOps practitioner and cybersecurity enthusiast from Bulgaria. I thrive at the intersection of automation, system administration, and ethical hacking. My technical journey has included setting up Dockerized environments, managing CI/CD workflows with GitHub Actions, and orchestrating services behind secure tunnels using Cloudflare.

I’ve worked on deploying and securing FileCloud with persistent storage in Docker, and developed a Redis-backed Flask application for visit tracking as part of a DevOps challenge. On the security side, I regularly use Kali Linux and tools like Nmap, Metasploit, Responder, and Bettercap to conduct penetration tests on intentionally vulnerable VMs like Metasploitable and JANGOW01.

This portfolio is a reflection of my technical growth, showcasing the tools and practices I use daily — from Git to containerization, from infrastructure-as-code to red-teaming tactics. I’m passionate about bridging the gap between DevOps and cybersecurity through hands-on learning and experimentation.

  • Linux & Scripting: Bash, systemd, apt, network config, SSH
  • Other: FileCloud deployment, Flask apps, Redis, reverse shells

DevOps Projects

Below are some of the projects I've worked on, demonstrating my practical experience in DevOps automation and cybersecurity:

  • Flask + Redis Visit Counter: A Python web app using Redis to count visits, deployed in Docker and managed with GitHub Actions for CI.
  • FileCloud Deployment: Self-hosted enterprise file sync service using Docker Compose, persistent volumes, and Cloudflare tunneling.
  • Penetration Test Labs: Hands-on testing on Metasploitable and hardened JANGOW01, using Nmap, reverse shells, and privilege escalation.
  • Athena Quiz App: Flask + PostgreSQL + NGINX project based on DevOps Essentials side project. [View repo]
